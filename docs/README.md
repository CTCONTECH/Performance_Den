# Performance Den Marketing Site

Modern, mobile-first marketing website for the Performance Den automotive performance brand.

## Structure

- index.html
- assets/css/styles.css
- assets/js/main.js
- assets/js/instagram-config.js
- assets/images/placeholder-car.svg

## Update Instagram posts

Edit the array in assets/js/instagram-config.js and replace the placeholder URLs with real Instagram post URLs.

## Run locally

Use any static server. Examples:

- Python: `python -m http.server 5500`
- Node: `npx serve .`

Then open http://localhost:5500 in your browser.
